# Vue Practise: Pokemon game

This repo is used for practising Vue 3 with basic concept

## Using
- Vue 3
- SCSS
- Typescript

## Install
1. Clone
   
   ```sh
   git clone https://github.com/HVTGiang/vue-pokemon-game
   ```
2. Install NPM packages
   
   ```sh
   yarn install
   ```
   or
   ```sh
   npm install
   ```
3. Run
   
   ```sh
   npm run dev
   ```
   or
   ```sh
   yarn dev
   ```
